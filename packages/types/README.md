# @types/stellar-hooks

Declaration-only package for [`stellar-hooks`](https://www.npmjs.com/package/stellar-hooks).

This package is generated from the canonical TypeScript source in the root `stellar-hooks` package. It exists for consumers that want to install declarations separately from the runtime package.

## Build

```bash
npm run build:types-package
```

The build emits declarations into `packages/types/dist`.
